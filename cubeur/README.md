# Cubeur Scierie

Application mobile de cubage pour scierie mobile.

## Déploiement sur Vercel

1. Importer ce repo dans Vercel
2. Framework preset : **Create React App**
3. Cliquer Deploy

## Fonctionnalités

- Saisie : produit, essence, qualité, dimensions, nb unités, volume grume
- Calcul automatique : volume unitaire, volume charge, rendement, perte
- Historique des charges avec totaux
- Export vers Google Sheets via Apps Script
